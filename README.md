## ⚠️ Critical Rules

### **NEVER:**
- ❌ Add features not in current week
- ❌ Use UIKit (SwiftUI only)
- ❌ Use Combine (async/await only)
- ❌ Add third-party libs without permission
- ❌ Skip error handling
- ❌ Hardcode strings/colors
- ❌ Force unwraps in production
- ❌ Hidden cancellation (App Store rejects)

### **ALWAYS:**
- ✅ Test in simulator after each feature
- ✅ Commit working code immediately
- ✅ Update PROGRESS.md daily
- ✅ Reference user pain points
- ✅ Honest with users (transparent pricing, easy cancel)
- ✅ Accessibility labels
- ✅ Dark mode support
